# Example 01: Plague Spread Damage-Over-Time

## Prompt
"A plague that spreads from enemy to enemy, dealing damage over time."

## Classification
- Types: `mechanics`
- Tier: mid/high persistent mechanic
- Suggested cost: `100,000` gold/credits

## Expected Runtime Additions
- DoT status effect system
- Spread/chaining logic between nearby enemies
- Default balancing values for this tier (damage, duration, spread cap)

## Notes
- This should be treated as a persistent mechanic with clear caps to prevent runaway scaling.
