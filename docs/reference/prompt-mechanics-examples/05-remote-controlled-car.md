# Example 05: Remote-Controlled Car Mechanic

## Prompt
"A remote-controlled car that can run over enemies and drive."

## Classification
- Types: `units`, `mechanics`, `ui` (if control HUD is required)
- Tier: interactive controllable entity
- Suggested cost: `20,000` units/credits

## Expected Runtime Additions
- Control binding mechanic for selecting/steering the car
- Car health and enemy collision damage rules
- Unit-vs-unit impact behavior
- Optional control/status HUD

## Notes
- This requires new player-control capability in addition to unit definition.
